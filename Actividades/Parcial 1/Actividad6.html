Siglas

OSI (Open Systems Interconnection)

Modelo conceptual que divide las funciones de comunicación en capas.

IP (Internet Protocol)

Protocolo que identifica dispositivos en una red mediante direcciones IP.

HTTP (Hyper Text Transfer Protocol)

Protocolo de transferencia de archivos en formato HTML (Páginas web).

FTP (File Transfer Protocol)

Protocolo utilizado para la transferencia de archivos entre sistemas.

TCP (Transmission Control Protocol)

Protocolo encargado de garantizar la entrega confiable de datos en una red.

UDP (User Datagram Protocol)

Protocolo sin conexión que permite enviar datagramas sin garantizar la entrega.

SSH (Secure Shell)

Protocolo utilizado para acceder de manera segura a equipos remotos.

SMTP (Simple Mail Transfer Protocol)

Protocolo utilizado para el envío de correos electrónicos.

POP (Post Office Protocol)

Protocolo utilizado para la recepción de correos electrónicos desde un servidor.

DNS (Domain Name System)

Sistema que traduce los nombres de dominio a direcciones IP.

ARP (Address Resolution Protocol)

Protocolo que mapea direcciones IP a direcciones MAC en una red local.

MAC (Media Access Control)

Dirección única de identificación asignada a una interfaz de red.

LAN (Local Area Network)

Red de área local que conecta dispositivos dentro de una misma área geográfica.
